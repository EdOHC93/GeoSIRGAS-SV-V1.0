DATUM Y SISTEMAS
Coordenadas en SIRGAS2000 (Sistema de Referencia Geocéntrico para las Américas), compatible con WGS84 a nivel de ±1cm. Altura respecto al nivel medio del mar usando modelo geoidal EGM2008 con correcciones para El Salvador (ondulación ≈ +25.4m zona central).
PROYECCIÓN CARTOGRÁFICA
UTM — Universal Transverse Mercator, Zona 16N (El Salvador, Guatemala oriental, Honduras occidental). Factor de escala k₀ = 0.9996. Falso Este = 500,000 m. Falso Norte = 0 m.
METODOLOGÍA DE PRECISIÓN
La sesión de 10 minutos (600 épocas a 1 Hz) permite reducir el error por promediado estadístico. La desviación estándar σ disminuye con √n observaciones. Con buena dilución de precisión (HDOP < 1.5) y ≥ 8 satélites de múltiples constelaciones, se alcanzan precisiones de 0.5–2.0 m en dispositivos consumer.

Para precisión centimétrica (≤10cm) se requiere corrección diferencial RTK via NTRIP usando la red CORS del IGN El Salvador o la red REGCEN de SIRGAS.
